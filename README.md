# computor-v1

> Résolution d'équations polynomiales — Projet 42

## Description

Programme PHP qui prend une équation polynomiale en entrée et affiche les solutions sur ℝ (degré ≤ 2 géré nativement, approche pour les degrés supérieurs).

Modules : parsing de l'équation, calcul matriciel, résolution et formatage de la sortie.

## Utilisation

```bash
php resolve.php "equation"
```

Fichiers principaux : `parse.php`, `polynome.php`, `matrice.php`, `resolve.php`.
